pub fn parse_with_encoding<'a>(input: &'a [u8], encoding_override: Option<EncodingRef>, use_charset: bool) -> Result<Parse<'a>, ()>

Convert a byte string in the application/x-www-form-urlencoded syntax into a iterator of (name, value) pairs.

Use parse(input.as_bytes()) to parse a &str string.

This function is only available if the query_encoding Cargo feature is enabled.

Arguments:

  • encoding_override: The character encoding each name and values is decoded as after percent-decoding. Defaults to UTF-8.
  • use_charset: The use charset flag. If in doubt, set to false.
